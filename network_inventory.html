📖 Router Guides / شروحات الراوتر

📶
Tenda Router
Change IP: Static ↔ Auto (DHCP)
1
افتح صفحة الراوتر
Open the router page
افتح المتصفح واكتب عنوان الراوتر: 192.168.0.1 أو tendawifi.com
Open browser and type: 192.168.0.1 or tendawifi.com
🖼 Click to add screenshot / اضغط لإضافة صورة
2
الدخول للإعدادات المتقدمة
Go to Advanced Settings
اضغط على Advanced أو إعدادات متقدمة من القائمة الجانبية
Click Advanced from the side menu
🖼 Click to add screenshot / اضغط لإضافة صورة
3
إعدادات الشبكة المحلية (LAN)
LAN Settings
اختار LAN Settings ثم ابحث عن IP Address
Select LAN Settings then find IP Address
🖼 Click to add screenshot / اضغط لإضافة صورة
4
تغيير نوع الـ IP
Change IP Type
لتغيير إلى تلقائي (DHCP): اختار DHCP وأحفظ
لتغيير إلى ثابت (Static): اختار Static IP واكتب الـ IP المطلوب
To Auto (DHCP): select DHCP and Save
To Static: select Static IP and enter the desired IP
🖼 Click to add screenshot / اضغط لإضافة صورة
⚠️ بعد الحفظ الراوتر يعيد تشغيل نفسه — انتظر دقيقة ثم تحقق من الـ IP الجديد من المودم
⚠️ After saving, router will reboot — wait 1 minute then verify new IP from the modem
🔴
Huawei Router
Change IP: Static ↔ Auto (DHCP)
1
افتح صفحة الراوتر
Open the router page
افتح المتصفح واكتب: 192.168.3.1 أو 192.168.100.1
Open browser and type: 192.168.3.1 or 192.168.100.1
🖼 Click to add screenshot / اضغط لإضافة صورة
2
الدخول لإعدادات WAN
Access WAN Settings
اضغط Advanced ثم WAN أو Internet Settings
Click Advanced → WAN or Internet Settings
🖼 Click to add screenshot / اضغط لإضافة صورة
3
تغيير نوع الاتصال
Change connection type
إلى تلقائي: اختار DHCP / Dynamic IP
إلى ثابت: اختار Static IP وأدخل: IP، Subnet Mask، Gateway
To Auto: select DHCP / Dynamic IP
To Static: select Static IP and enter: IP, Subnet Mask, Gateway
🖼 Click to add screenshot / اضغط لإضافة صورة
⚠️ كلمة السر الافتراضية للـ Huawei الجديد عادةً مكتوبة على ملصق الجهاز من الأسفل
⚠️ Default password is usually printed on the label at the bottom of the device
🌐
Google WiFi / Nest WiFi
Change IP: Static ↔ Auto (DHCP)
1
افتح تطبيق Google Home
Open Google Home App
Google WiFi يُدار عبر تطبيق Google Home على الجوال — لا يوجد صفحة ويب
Google WiFi is managed only through the Google Home mobile app — no web interface
🖼 Click to add screenshot / اضغط لإضافة صورة
2
إعدادات الشبكة المتقدمة
Advanced Network Settings
من التطبيق: WiFi ← Settings ← Advanced Networking
In app: WiFi → Settings → Advanced Networking
🖼 Click to add screenshot / اضغط لإضافة صورة
3
تغيير الـ IP
Change IP
اضغط على WAN ثم:
إلى تلقائي: اختار DHCP
إلى ثابت: اختار Static IP وأدخل البيانات
Tap WAN then:
To Auto: select DHCP
To Static: select Static IP and fill in the details
🖼 Click to add screenshot / اضغط لإضافة صورة
⚠️ Google WiFi لا يدعم تغيير IP من المتصفح — التطبيق إلزامي
⚠️ Google WiFi does not support IP changes via browser — the app is required
