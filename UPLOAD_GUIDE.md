## 🚀 خطوات الرفع على GitHub:

### الطريقة 1: من الويب مباشرة (الأسهل)

1. **اذهب إلى GitHub:**
   ```
   https://github.com/asmd2268/asmd1-network
   ```

2. **اضغط على folder اللي تبيه** (مثل اسم الـ repo)

3. **اضغط على "Add file" → "Upload files"**

4. **اسحب الملفات من هنا:**
   ```
   network_inventory_fixed.html
   network_inventory_supabase.html (اختياري)
   ```

5. **اضغط "Commit changes"**

6. **Vercel سيعيد البناء تلقائياً** ✅

---

### الطريقة 2: من الكمبيوتر (لو معك Git)

```bash
# 1. Clone الـ repo
git clone https://github.com/asmd2268/asmd1-network

# 2. انسخ الملفات الجديدة للـ folder
cp network_inventory_fixed.html ./repo/
cp network_inventory_supabase.html ./repo/

# 3. أضفها لـ Git
git add network_inventory_fixed.html
git add network_inventory_supabase.html

# 4. Commit
git commit -m "Add fixed password authentication"

# 5. Push
git push origin main
```
